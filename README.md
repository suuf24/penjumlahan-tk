# Belajar Berhitung Anak TK

Aplikasi web sederhana untuk latihan berhitung anak usia pra-TK/TK.  
Aplikasi ini dibuat dalam **1 file HTML saja** dan dapat langsung dijalankan di browser tanpa instalasi tambahan.

## Deskripsi

Aplikasi ini membantu anak belajar berhitung dasar dengan angka maksimal sampai **9**.  
Soal dibuat secara acak menggunakan JavaScript, sehingga setiap kali anak menekan tombol **Mulai Lagi**, soal akan berubah.

Aplikasi cocok untuk:
- Anak pra-TK
- Anak TK
- Anak yang baru mulai belajar angka
- Latihan berhitung dasar di rumah atau sekolah

## Fitur Utama

- Latihan berhitung angka 1 sampai 9
- Soal muncul satu per satu
- Total 10 soal dalam satu sesi
- Soal dibuat secara acak
- Pilihan jawaban acak
- Menggunakan tombol pilihan jawaban
- Feedback ramah anak
- Skor sederhana
- Hasil akhir dengan pesan motivasi
- Tampilan cerah, lucu, dan kids friendly
- Responsive untuk HP, tablet, dan laptop
- Tidak menggunakan library eksternal
- Tidak menggunakan framework

## Jenis Soal

Aplikasi dapat membuat beberapa jenis soal, yaitu:

1. Menghitung jumlah benda  
   Contoh:  
   Ada berapa gambar ini?  
   🍎 🍎 🍎

2. Penjumlahan sederhana  
   Contoh:  
   3 + 2 = ?

3. Memilih angka sesuai jumlah gambar  
   Contoh:  
   Pilih angka yang sama dengan jumlah gambar!  
   ⭐ ⭐ ⭐ ⭐
